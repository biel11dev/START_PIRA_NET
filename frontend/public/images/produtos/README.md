# Pasta de Imagens dos Produtos

Adicione as imagens dos produtos nesta pasta.

## Como usar:

1. Adicione a imagem aqui (ex: `agua-cristal.png`)
2. No painel admin, use o caminho: `/images/produtos/agua-cristal.png`

## Recomendações:

- Formato: PNG ou JPG
- Tamanho: 500x500px (quadrado)
- Nome: use-kebab-case.png
- Tamanho máximo: 500KB

## Exemplo:

```
/images/produtos/coca-cola-2l.png
/images/produtos/agua-mineral.png
/images/produtos/pizza-calabresa.png
```
